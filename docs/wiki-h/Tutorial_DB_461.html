Fix tutorial
Decedra Willham - gossip text

Revision: DB 461
Issue: #000184
Difficulty: easy
Category: NPC

Problem: Decedra Willham was not displaying proper greeting text when spoken to.

Step-by-step solving

Step 0: Find what effect we want to achieve; Fast look at wowpedia.org gives us quote NPC is supposed to say:

I do so enjoy sitting out here, listening to the sounds of the Great Sea. Oh, what? Yes, there is a ship that will take you to Menethil Harbor. It docks here from time to time.


Step 1: Check if there is any similar text in database; we choose distinctive part like "Great Sea" and run a query on world database: 

SELECT * FROM `npc_text` WHERE `text0_0` LIKE "%Great Sea%"

Query returns us 4 results, none of them is quote we look for.


Step 2: Add quote to database; we need to choose any free text id, query 

SELECT `ID` FROM `npc_text`

gives us text ids that are already used, we take any free one, in this example 3. Now we need to assign our quote to this id 

INSERT INTO `npc_text` (`ID`, `text0_0`) VALUES (3,"I do so enjoy sitting out here, listening to the sounds of the Great Sea. Oh, what? Yes, there is a ship that will take you to Menethil Harbor. It docks here from time to time.");


Step 3: Assign quote to npc; gossip texts are assigned to individual creature not whole template, so we need to find GUID of Decedra Willham 

SELECT `guid` FROM `creature` WHERE `id` = 12716;

we get one result: 8464. So there is just one last thing to do, NPC with GUID 8464 should use text 3 when spoken to 

INSERT INTO `npc_gossip` VALUE (8464,3);

Summary

Query fixing this problem: 

INSERT INTO `npc_text` (`ID`, `text0_0`) VALUES (3,"I do so enjoy sitting out here, listening to the sounds of the Great Sea. Oh, what? Yes, there is a ship that will take you to Menethil Harbor. It docks here from time to time.");
INSERT INTO `npc_gossip` VALUE (8464,3);
