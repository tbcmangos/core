Fix tutorial
Screecher Spirits

Revision: 8075
Issue: #000262
Difficulty: easy
Category: Scripts

Problem: in quest Screecher Spirits player was able to use Yeh'kinya's Bramble to one dead vale screecher multiple times.

Step-by-step solving

Step 0: Similar problem was previously encountered with other "use item on body" quests, like infamous Further Conversions on Isle of Quel'danas, so we can check how it was solved.


Step 1: Where to begin; Problem is about item script - when item which has its script the script is used before actual action happens. In this case between checking trivial things (does player have this item, can he use it, is it on cooldowns) and actualy casting the spell core goes through item script function (set by `ScriptName` field in item template). Item scripts are stored in item_scripts.cpp.


Step 2: Bright ideas; "our" script is ItemUse_item_yehkinyas_bramble(...) when just one scroll down we can see ItemUse_item_specific_target(...) which is used for many similar situations. We prefer doing things globally, so let's try to fix this by using ItemUse_item_specific_target(...)


Step 3: We have tool, let's use it; ItemUse_item_specific_target(...) is a simple and fine-working machine, it allows item to be used only on creatures with entries in cEntry[] with optional check if creature is dead/alive. So we simply add another line in main switch: 

case 10699: cEntry[0] = 5307;  cEntry[1] = 5308;  targetState = T_DEAD; break;

so this item can be only used on vale screechers' corpses.


Step 4: Cleanups; so now we can remove useless ItemUse_item_yehkinyas_bramble(...) we should also remove this script from manager. AddSC_item_scripts() registers each item script in manager, from this function we remove 

newscript = new Script;
newscript->Name="item_yehkinyas_bramble";
newscript->pItemUse = &ItemUse_item_yehkinyas_bramble;
newscript->RegisterSelf();


Step 5: Finish; now we just need to force quest item to use other script, we simply modify `ScriptName` in item template 

UPDATE `item_template` SET `ScriptName`="item_specific_target" WHERE `entry` = 10699;

Summary

So we need to do just two simple things: switch item to use good script and tell script what targets this item needs.
