HellGroundCore is a Wow server emulator. It is derived from MaNGOS and Trinity, and is based on the code of these projects.

For further information on the HellGroundCore project, please visit our project repository at http://emu.hellground.net

SQL files to create the database can be found in the sql directory. Files to update your database from an older revision/version can be found in the sql/updates directory.
