This dbc contains information about emotes that can be played on NPCs (for scripting,etc). To view the emote, use the .npc playemote <emoteid> command. Playing Emote 0 will clear an emote that is looping.

Structure

Structure as of patch 2.4.3.

Field No Name Type Comment
1 ID Int Emote ID
2 InternalName String System Name for the Emote
3 AnimationData Int Reference to the entry from AnimationData.dbc played
4 Flags Bitmask 0x8: Talk, 0x10: Question, 0x20: Exclamation, 0x40: Shout, 0x100: Laugh. These are used for the chatting emotes.
5 IsLooped Int Theory: 0 = perform emote then revert; 1 = loop (emotes); 2 = loop states (ie stun). Actually, at some point in the code, a sound is only played if this is set to 2. It has been noted that some emotes will loop even if this value is 0.
6 Hold Int Hold the last frame of the animation. Only set for kneel, dead, sleep, at_ease and sit.
7 SoundEntry Int A sound (from SoundEntries,dbc) that is played on this emote.
