Preface

This overview is to help minimize the time it takes to implement an HellGroundCore server from the viewpoint of starting with nothing. Well, some things.
HellGroundCore is a server engine for World of Warcraft 2.4.3-8606, that was the last update before the 3.x.x updates in preparation for the 2nd expansion (or add-on), "The Wrath of the Lich King". You can think of it as the final version of "The Burning Crusade".

Installation from Zero

The first thing you will need to make a useful HellGroundCore server is a copy of World of Warcraft. That software creates the client, and is the bulk of the game's data. The client has all the graphics and maps and magic (literally in some cases). Because HellGroundCore is capable of dealing with (if not being dependent on) "The Burning Crusade" World of Warcraft add-on, that should be your second thing.
The remaining things are dependent on your goals and available resources. You may be attempting to make a publicly available server that is remotely administered from a game server farm. You may just be trying to set up a single computer to run the client and the server within itself. In either case, and most of the cases in between, the steps are similar.

To completely install and configure HellGroundCore as a server you will perform the following steps:

  1. Install the client (fresh)
    1. Install the add-on (fresh)
    2. Update the client to 2.4.3-8606 (exactly)
  2. Obtain the HellGroundCore source repository
    1. Compile the source
    2. Install the executables
  3. Make the required vmaps from the client
  4. Create an sql server
    1. Create databases for the sql server
    2. Configure the databases
  5. Configure the server
  6. Configure the client to attach to the server


And once these tasks are complete you should have a server capable of handling TBC WoW clients.

Installation of Client

First you need an installation of the client patched to 2.4.3-8606. You can check you current version by loading up WoW and looking at the bottem left corner of the screen. If you already have one, great! If like most people you have a newer patch or just want to start with a fresh copy do the following.

  1. install the original World of Warcraft from disc (6 CD-ROM/1 DVD) (as normal or use the alt install below)
  2. install The Burning Crusade expansion pack (4 CD-ROM/1 DVD) (as normal or use alt install below)


Alternate Install (still requires original media)
You can minimize the install time or help multiple client installs over network (LAN party) by creating a folder for the install set (WoW or TBC) and copying (1) the whole first disc, and then (2) the MPQ files from each of the remaining disc. I'm pretty sure this works for the WoW install and I know it works for TBC.
You then just run the Installer.exe (or appropriate) from the folder and you no longer have to swap discs - the installer will automatically load from the mpq files within the directory.

Also look at the list of configuration information in the above for DB configuration instructions. Do those as well.

Configure Client

To configure the client to attach to you new server, edit the realmlist.wtf in the C:\Program Files\World of Warcraft folder so that it contains only:

set realmlist <yourip>

Where <yourip> is the ip address of the server (like 127.0.0.1 or similar). It will not have <..> brackets.
