Each unique pool is defined in this table.

`pool_template` table rev. 7933
Field Type Null Key Default Extra
entry mediumint(8) unsigned PRI NO 0
max_limit int(10) unsigned NO 0
description varchar(255) YES

entry

The pool ID. This is an arbitrary number that is only used to link the gameobjects and creatures in this pool.

max_limit

This is the maximum number of objects that should be spawned in this pool.

description

Field describes the basic information about what the pool refers to. Example: Snarlflare (14272)
